# Changelog

## 2.0.0 - 2026-03-31

- Nova abordagem com downloads reais iniciados pelo proprio `Transfira-now` usando `DownloadManager`.
- Priorizacao de jornadas do proprio app para aumentar a elegibilidade a surfaces promovidas no Android 16.
- Adocao de `NotificationHelper` dedicado, com icone dinamico de progresso em bitmap e tentativa de promoted ongoing/live update.
- Separacao entre downloads do app e downloads detectados de terceiros na interface.
- Mantido fallback confiavel para terceiros via lock screen + notificacao continua.
- APK publico atualizado para distribuicao.

## 1.0.0 - 2026-03-31

- Criacao inicial do app `Transfira-now`.
- Interface Material You inspirada no visual do Nowbar Meter.
- Monitoramento de downloads por notificacoes.
- Inicio de downloads reais pelo proprio app com `DownloadManager`.
- Servico em segundo plano com notificacao continua.
- Tela de bloqueio com informacoes publicas de progresso.
- Icone, splash screen e identidade visual definidos.
- APK debug gerado com sucesso.
